How I made Favorites

I created an iframe that takes a spotify link as a source, and then created a form with a label, input line, and submit button. When the button is pressed, the changeSong function in the javascript file is called.

I then created the changeSong() function to grab the new song's link and attach its id to the original spotify link after after removing the original id.
